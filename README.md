# StudyRoom

A collaborative study room application for students and study groups.

## Features

- **Authentication**: Login, Registration, Onboarding
- **Home**: Dashboard with quick access to all features
- **Groups**: Create and manage study groups
- **Meetings**: Schedule and manage study sessions
- **Chat**: Real-time messaging within groups
- **Calendar**: View and manage meetings
- **Notifications**: Stay updated with important alerts
- **Search**: Find groups and members
- **Profile**: Manage your profile and settings

## Tech Stack

- **Language**: Kotlin
- **Platform**: Android
- **Backend**: Firebase (Firestore, Authentication)
- **Architecture**: MVVM
- **UI**: Material Design

## Getting Started

### Prerequisites

- Android Studio (latest version)
- Android SDK
- Firebase account

### Installation

1. Clone the repository
2. Open in Android Studio
3. Add your `google-services.json` file in `app/`
4. Build and run

## Project Structure

```
app/src/main/java/com/projet/studyroom/
├── auth/           # Authentication activities
├── chat/           # Chat functionality
├── groups/         # Group management
├── home/           # Main dashboard
├── meetings/       # Meeting management
├── notifications/  # Notifications handling
├── profile/        # User profile
└── search/         # Search functionality
```
